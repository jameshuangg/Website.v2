Easy Books

Currently still a work in progress. Easy Books is my first attempt at utilizing open source api. Hopefully, it will also provide reading analytics, so I can improve my habits.

TypeChef

Type Chef is an interactive typing web application made during UofT Hacks 2017. The goal of this app was to solve the problem of today's generation. Not many people are proficient at typing and have a hard time cooking. We aimed to solve this problem by providing an application that trains typing and cooking knowledge through repetition. I was responsible for the frontend implementation. Github link can be found here.
